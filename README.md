# android-curl
so支持arm64、x86_64架构，可以在真机或者模拟器上运行。
编译功能包括curl+mbedtls+nghttp2.
独立的aar module，可以直接引入项目使用，结合app中的封装demo，可以快速集成进入项目中。
更多请见文档：https://www.jianshu.com/p/895a4e5052e2
